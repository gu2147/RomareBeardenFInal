How to Use the Canvas Tool

  • Brush: Allows you to paint on the canvas. You can choose different colors.
  • Eraser: Lets you erase parts of your painting to correct mistakes or create effects.
  • Fill Color: Enable this to fill shapes with the selected color instead of just drawing the outlines.
  • Shape Tools (Rectangle, Circle, Triangle): Use these to draw the respective shapes.
  • Imae Tools: Use these to draw the respective Bearden Image. This will work like a stamp and can be erased with the eraser tool.
  • Size Slider: Adjust the size of your brush, image, or eraser to be larger or smaller.
  • Color Picker: Select the color you wish to paint with.
  • Clear Canvas: Clears all content from the canvas, giving you a fresh start.
  • Save Image: Save your artwork as an image file to your device.
